Web Page Elements

When you view a web page in your browser, you are seeing the interprted code that you type into your text editor. Web pages are made up of text which is called "Marked Up", or formatted using Hyper Text Makeup language or HTML codes, known as tags, are placed around the text that you want to format. In most cases, when formatting text, you will have both an opening and closing tah to indicate where the formatting should start and where it should end.

Content on a web page is contained inside elements. There are basically two types of elements, block and inline. This content is being written inside a paragraph element, which is a block element because ther will be a space above and below th paragraph essentially blocking if off from other content. However this content is in a strong element which is inline. Inline elemnts do not separate their content. Block elements will effect a pages layout, where inline elements will not.

There are all different types of elements that go to make up a web page. The elemets used on this page is the HTML, which is also called the root element, the head element, body element, heading element, paragraph element, and the strong inline element. The elements use on this page, other than the strong element, are the minimum that any web page should include.
